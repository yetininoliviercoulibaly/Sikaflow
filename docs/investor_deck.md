## 1. Le Problème (The Pain)

Dans l'industrie de la nuit (Nightlife) et de l'événementiel :

| Problème                   | Impact                                                       |
| :------------------------- | :----------------------------------------------------------- |
| **Perte de Cash**          | 15-20% du CA disparaît dans les petites dépenses non tracées |
| **Outils Inadaptés**       | Les gérants refusent QuickBooks/Sage sur le terrain          |
| **Données Inexploitables** | Tout passe sur WhatsApp mais reste non structuré             |
| **Billetterie Fragile**    | Fraude, double utilisation, pas de traçabilité               |

---

## 2. La Solution : Event-Pilot

Event-Pilot n'est pas une nouvelle application. C'est une **IA superposée à WhatsApp**.

### La Magie (Secret Sauce)

1. **Multimodalité Totale** : Texte, vocal, photos de reçus
2. **IA Générative (Gemini)** : Comprend le jargon local (Nouchi, Joual)
3. **Reporting Automatique** : P&L quotidien sans saisie manuelle
4. **Billetterie Intégrée** : Création, vente, scan via WhatsApp

---

## 3. Fonctionnalités Clés

### 🎯 Gestion Opérationnelle

- Saisie multimodale des dépenses (texte/vocal/photo)
- Incidents tracés et horodatés
- Rapports PDF instantanés

### 🎫 Billetterie WhatsApp-Native

- Création d'événements en langage naturel
- Génération de billets avec QR codes sécurisés
- Scan de validation par photo
- Feedback post-événement automatisé

### 🎓 Onboarding Intelligent

- Tutoriel interactif en 5 étapes
- Détection automatique des actions (Event-Driven)
- Accompagnement personnalisé

| Composant | Technologie                      |
| :-------- | :------------------------------- |

| Database  | PostgreSQL + MikroORM            |
| Queue     | BullMQ + Redis                   |
| IA        | Google Gemini (Vision + Text)    |
| Paiements | Stripe + Wave                    |

**Moat technique** :

- State Machine Multi-Organisation
- OCR & Audio entraînés sur données africaines
- Offline-First via WhatsApp
